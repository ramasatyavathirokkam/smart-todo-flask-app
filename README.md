# Smart-todo-flask-app
A simple and smart task management web application built using Flask. This app helps users organise their daily tasks efficiently with features like priority levels and due date tracking.
# Features
- Add, delete, and mark tasks as completed
- Set Task priority(high, Medium, Low)
- Add due dates for better planning
- Persistent storage using JSON
- Clean and responsive user interface
# Tech Stack
- Python(Flask)
- HTML, CSS
- Jinja Templates
- JSON(for data storage)
# How It Works
- Users enter task through a form
- Data is sent to the Flask backend
- Tasks are stored in a JSON file
- The updated task list is rendered dynamically on the webpage
# Future Improvements
- User authentication(login system)
- database integration(SQLite/MySQL)
- Task editing feature with UI
